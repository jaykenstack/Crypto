# Wallet Scanner Simulator

A Python-based simulator that generates and scans mock blockchain wallet addresses to demonstrate wallet scanning concepts.

## Features

- Generate mock blockchain addresses (Ethereum, Bitcoin, Solana formats)
- Simulate scanning addresses for balance and transaction history
- Multi-threaded scanning for performance
- Real-time statistics and progress tracking
- Console-based UI with colored output
- Configurable simulation parameters
